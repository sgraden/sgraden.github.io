The Gidget Project

Gidget is a game designed to teach computer programming concepts through debugging puzzles. The project was started by Michael Lee for his Ph.D. dissertation and as an opportunity to research new methods of teaching programming through online resources and, more specifically, games. Michael put in enormous amounts of effort to design a game which is gender neutral and engaging to all children.

The Games Premise

Gidget is a game designed to teach computer programming concepts through debugging puzzles. Gidget the robot was damaged on its way to clean up a chemical spill and save the animals, so it is the players' job to fix Gidget's problematic code to complete all the missions.

State of the Project

During the Summer of my Junior year (2014) at the University of Washington I, and 2 others, were accepted as Research Assistants for Michael and his Ph.D. project. This was about the third year Michael's project had been in progress and it was to be released by the end of the summer.

The role of the research assistants was to help finish the project in any way we could. This means we were to research, design, develop, and test the game continously throughout the summer. I took the opportunity to focus on UX design and front-end development.

When I came onto the project, the core infrastructure of the game was already in place. The user was able to enter code, run it, progress through levels, log in/out, etc. On the backend there were a lot of hacks and inefficiencies in the code, so one of the Research Assistants worked on that while I took the opportunity to improve and refine the front-end from a functional UI to a more enjoyable and easy to use UI.

My Contributions

Here is a list of some of the work I did on Gidget over the summer:
  • Transformed the start page from a plain grey box to the more playful UI it is today
  • Created the UI and backend for a user to manage and access custom levels
  • Refined the "Command Dictionary" visuals and interactions
  • Added an improved and interactive tutorial to help explain the UI using Intro.js, a lightweight plugin
  • Connected AnswerDash and created a list of potential Q&A to quickly help the users if they were ever stuck
